Scoring methodology
Score = token usages / (token usages + violations). Only property declarations in relevant categories are scanned: spacing properties for space/size tokens, color properties for hex colors, border-radius for radius, font-size for typography. Values like 0 are only flagged in spacing/radius contexts, not in z-index or opacity.
Space tokens dominate violations
Space/size violations account for 100% of all violations. Values like 0, 1rem, 0.5rem in margin/padding/gap should use var(--luna-space-*) equivalents. This is the single highest-impact migration target.
