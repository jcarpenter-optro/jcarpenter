Priority Recommendations
1
Invest in OpsAudit fieldwork UX as a dedicated improvement program
The "paper cuts" described across 307 complaint records and multiple NPS detractor comments (including a 5-year customer) reflect accumulated UX debt: no rich text in workpapers (a request since 2011), workstep linking failures, non-intuitive navigation, and unsolicited UI changes. Establish a dedicated OpsAudit UX working group with a commitment to: (a) no unrequested UI changes without opt-in
307 complaint records + long-tenure NPS detractors
2
Automate the audit planning lifecycle end-to-end
Audit planning automation is the #1 improvement request (466 records, 453 Gong calls). Priority deliveries: (a) auto-populate audit plan from AURA risk scores, (b) risk-based coverage recommendations with AI (145 records), (c) dynamic reprioritization when risk scores change mid-cycle, and (d) version-controlled plan changes. This is also the core Accelerate upsell narrative— tying planning

Deliver native parent-child auditable entity hierarchy support
Difficulty handling complex hierarchies (263 complaint records, 235 Gong calls, 8 community votes) is a recurring enterprise deal qualifier. Multi-entity, multi-country, and multi-business-unit audit teams need native parent-child entity structures with hierarchical rollups for reporting. This is explicitly named as a TeamMate+ feature that customers expect in a modern audit platform.
